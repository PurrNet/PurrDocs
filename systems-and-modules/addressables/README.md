# Addressables

PurrNet offers out of the box addressables support for Unity. It handles the usage fully asynchronous as well, so it doesn't bake all the addressables initially, also allowing for more flexibility and more optimized memory management.

{% hint style="info" %}
Addressables support is currently a young feature, and is likely to be improved in the future. Let us know if anything doesn't work as intended, and we'll be right on it!
{% endhint %}
